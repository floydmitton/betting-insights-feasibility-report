# Public Website (5–10 Pages)

All pages built in **Next.js**, fully mobile-responsive across 375px, 768px, and 1280px breakpoints. SEO-optimised with unique meta titles, meta descriptions, Open Graph tags, and a sitemap.xml.

**Client supplies:** All copy and brand assets (logo, colours, fonts) at kickoff.

---

## Home Page

- Hero section with headline, subheadline, and primary CTA button linking to signup
- 3-step "how it works" explainer
- Stats bar (e.g., sportsbooks covered, average monthly profit figure, active members)
- Feature highlights grid (3–4 features with icons)
- Social proof section (testimonials or placeholder slots for early reviews)
- Secondary CTA section above the footer

## How It Works Page

- Full explainer of the matched betting process broken into numbered steps
- Worked example showing a real bonus conversion calculation:
  > Example: $200 DraftKings free bet converted to approximately $160 cash
- FAQ accordion (minimum 8 questions)
- Responsible gambling disclaimer and 1-800-GAMBLER reference

## Features Page

- Detailed breakdown of every tool available on the platform with screenshots or Figma mockups
- Tier comparison showing which features are Free vs Basic vs Premium
- CTA to start free

## Pricing Page

- 3-column tier comparison table (Free, Basic, Premium) with feature checklist per tier
- Monthly and annual toggle (annual pricing shown at a discount)
- Each tier has a dedicated signup CTA button routing directly into the correct Stripe checkout flow

## FAQ Page

Minimum 15 questions covering:
- How the platform works
- What sportsbooks are supported
- How to get started
- Account restrictions
- Responsible gambling
- Billing and cancellation

## Blog / Resources Hub

- Archive page listing posts by category (Getting Started, Strategy, State Guides, Sportsbook Reviews)
- Individual post template with proper formatting
- **No content writing included** — structure and templates only
- Client populates content post-launch

## Contact / About Page

- Company background
- Team section (placeholder if needed)
- Contact form that emails to a nominated client address

---

## Developer Feasibility Notes

**Effort: LOW** — This is standard Next.js marketing site work with well-established patterns. SSR/SSG for SEO is built into Next.js. Responsive design with Tailwind CSS is straightforward. The only variable is how much design iteration the client requires.

**Estimated time:** 3–5 days of development after designs are approved.
